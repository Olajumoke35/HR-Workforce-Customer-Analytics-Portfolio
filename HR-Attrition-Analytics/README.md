# Data Analytics Portfolio – HR, Workforce, and Customer Operations

This portfolio showcases projects focused on HR analytics, workforce planning, and customer operations analytics.

The goal of these projects is to demonstrate how data analysis can support better workforce decisions and operational performance.

## Tools

SQL  
Excel  
Power BI  

## Projects

### HR Attrition Analytics
Analysis of employee turnover patterns and retention risk.

### Workforce Capacity Planning
Workforce productivity and staffing gap analysis.

### Customer Service Analytics
Customer support performance analysis including response time and resolution efficiency.

### Recruitment Pipeline Analytics
Recruitment funnel analysis and hiring efficiency metrics.

### Employee Performance Analytics
Employee productivity and performance distribution analysis.
